# Jeopardy: Data Security Edition

A corporate-friendly, single-page, HTML/CSS/JS Jeopardy-style game focused on data security. Includes 4 teams, 5 categories × 5 clues, and Final Jeopardy with wagers.

## Play
Open: `https://<your-username>.github.io/jeopardy-data-security/`

## Features
- 4 teams (editable names), live scoring, persistent state
- 5 categories × 5 clues, themed on data security
- Final Jeopardy with wagers and reveal scoring
- Accessible, responsive, no external dependencies

## Customize
Edit the `GAME` object in `index.html` to change categories, clues, and the final question.

## Publish (GitHub Pages)
1. Create repo and add `index.html`.
2. Enable Pages: Settings → Pages → Deploy from branch (`main` / root).
3. Open the published URL.
